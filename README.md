# MasterE

# 프로젝트 디렉터리 구조
```
MasterE           
├───Document        # MasterE 프로젝트의 문서를 모아놓은 디렉터리입니다.
├───Extensions      # MasterE 프로젝트의 Chrome-Extension 디렉터리입니다.
│   ├───images
│   ├───page
│   └───scripts
├───Backend         # FastAPI 백엔드 파일들을 모아놓은 디렉토리입니다.
├───Frontend        # React 프론트엔드 파일들을 모아놓은 디렉토리입니다.
└───MySQL           # MySQL 디렉터리입니다.
```
